What I Do

Graphic Designe

Adobe Photoshop과 Illustrator를 사용하여 다양한 배너와 로고아이콘 및 웹 사이트 개발 전 시안 작업을 할 수 있습니다.

Development

HTML5를 사용하여 웹 사이트의 구조를 잡고 CSS3로 레이아웃 배치 및 구체적인 디자인을 잡을 수 있습니다. 그리고 Javascript & jQuery 를 사용하여 웹 페이지를 유연하게 조작할 수 있게 합니다.

Responsive Web

Bootstrap을 메인으로 레이아웃을 잡고 CSS의 Media query 기능을 사용하여 데스크탑부터 모바일까지 전기종 호환되는 반응형 웹 사이트를 만들 수 있습니다.

Progressive Web App

뛰어난 브라우저의 성능을 이용하여 앱보다 접근성이 좋으며 설치도 가능한 웹 앱을 만들 수 있습니다.
